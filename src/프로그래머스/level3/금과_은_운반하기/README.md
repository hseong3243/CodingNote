## 핵심 포인트

- 이분 탐색
- 이분 탐색의 max 값이 400조 정도로 굉장히 크게 잡히는 문제이다.
- 아이디어를 도출하는 것은 어렵지 않으나 `금을 우선적으로 운송했을 때 운송할 수 있는 금의 양`과 `은을 우선적으로 운생했을 때 운송할 수 있는 은의 양`을 각각 `Gmax`와 `Smax`라 할 때, `a <= Gmax`이고 `b <=Smax`이면 `a+b <= Gmax + Smax`이다. 라는 증명을 해내지 못했다.
